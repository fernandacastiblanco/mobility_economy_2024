# mobility_economy_2024- Sprint 5

Este repositorio contiene el análisis realizado durante el Sprint 5 del caso mobility_economy_project.

El dataset `tomtom_traffic.csv` incluye 1004464 registros de viajes realizados para diferentes ciudades en distintos países del mundo.
El dataset `oecd_city_economy.csv` incluye 30 regsitros por ciudad en distintos países con dat9os macroeconómicos.

## 📂 Contenido del repositorio

- `S5 ladb_mobility_economy_project_student.ipynb`
  → Notebook principal con exporación, normalización y representación gráfica de datos.

## 🧠 Objetivo del análisis

Evaluar cómo la movilidad urbana se relaciona con la productividad económica en las principales ciudades latinoamericanas. Para ello se trabaja con datos reales de TomTom Traffic Index y OECD Cities.
